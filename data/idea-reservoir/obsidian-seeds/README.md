# Obsidian Seeds

This folder is the curated Obsidian export/sync area for Proverbs Daily idea material.

Fred's full Obsidian vault, FlynchMind, remains outside this repository and should not be committed here.

Only Proverbs Daily relevant markdown files should be copied or synced into this folder.

Keep Obsidian markdown links, tags, and backlinks when present. These may become useful signals for future idea matching, source tracing, and dashboard suggestions.

Obsidian Seeds are suggestive creative inventory, not authoritative published content.

Future processing may convert selected markdown notes into structured `ideaPool` entries.
