## Manual Workflow

1. Capture or move Proverbs Daily related seeds into the Obsidian source folder.
2. Periodically copy selected `.md` files into the project intake folder.
3. Commit the copied files to Git only when they are intentionally part of the Proverbs Daily production system.
4. Do not commit the full FlynchMind vault.
5. Do not delete ideas just because they are not chosen.
6. Later, processed ideas can be converted into structured entries in `src/ideas/ideaPool.json`.

## Design Principle

The Idea Reservoir should preserve uncertainty. Not every seed needs an immediate verse match.

Wisdom often arrives as a connection before it arrives as a conclusion.
