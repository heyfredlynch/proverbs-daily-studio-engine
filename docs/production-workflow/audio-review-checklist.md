# Audio Review Checklist

Use this checklist for each Week Two audio file after ElevenLabs generation.

## Checks

- Scripture reference correct
- Verse text correct
- Title correct
- Voice sounds natural
- No weird pacing
- No mispronounced names
- No internal labels leaked
- Prayer has exactly 10 words
- Remember line is clean and quotable
- Challenge is clear
- Outro included only where intended

## Review Status

Mark one:

```text
AUDIO_APPROVED
NEEDS_REGEN
NEEDS_SCRIPT_FIX
```
