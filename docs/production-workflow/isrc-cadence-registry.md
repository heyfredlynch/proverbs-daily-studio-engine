# ISRC and Cadence Registry

ISRC identifies a specific sound recording. It is not a song title, script, episode idea, project, or composition identifier. It points to a particular recorded master.

Each final mastered audio recording may need its own ISRC. Previews, alternates, remasters, mastered versions, and music-backed spoken-word versions may need separate review before any code is assigned.

Proverbs Daily, Get Strong, Sacred Hum, and Living Book audio can all be tracked in Cadence under Sacred Studios recording metadata. This registry is the local source of truth for pending and assigned sound recording identifiers.

The registry does not auto-issue codes. ISRC assignment remains manual until Fred explicitly approves a recording for assignment.

Sacred Studios prefix:

```text
QT8YT
```

Example stored format:

```text
USQT8YT2600001
```

Human-readable format:

```text
US-QT8YT-26-00001
```
