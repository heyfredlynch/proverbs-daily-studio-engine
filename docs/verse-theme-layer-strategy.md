# Verse Theme Layer Strategy

The Verse Theme Layer supports planning and matching by giving the engine lightweight, searchable signals for individual Proverbs verses.

Verse themes are not replacements for scripture. They are suggestive planning metadata only. Final script generation must always use the actual scripture text, not this theme layer, as the source for biblical reflection.

This layer helps the planning engine discover meaningful connections between unused verses and available Idea Reservoir seeds. It gives Idea Matching a path beyond broad chapter-level matching without forcing premature conclusions.

## Current Scope

Version 0.1 includes seed theme data only for the candidate verses currently appearing in the weekly plan.

The structure is designed to scale to all Proverbs chapters and verses later.

## Rules

- Keep summaries concise.
- Do not quote scripture text.
- Do not write devotional content.
- Treat themes as suggestive, not authoritative.
- Preserve flexibility for future interpretation.

## Future Sources

Future versions may derive or refine themes from:

- the Proverbs Daily canonical archive
- the Proverbs Daily app database
- scripture sources
- Fred's editorial review
- manually curated verse study notes
